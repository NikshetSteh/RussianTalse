Название проекта: Russian_tales
Платформа: Android
Среда разработки: Android studio

Описание проекта:
    Приложение: "Энциклопедия русских сказок".
 Приложение включает в себя не только описание сказок, но и викторины,
 которые помогут оценить знания сказок. Возможность дополнить энцикло-
 педию другими сказками.

Техническое описание:
    Исходные файлы представляют из себя стандартный проект Android studio, находить в папке: "app"
    Готовый файла формата .apk находятся в папке: "build".
    
    С точки зрения пользователя:
        При входе в приложение появляется пустой экран и четыре кнопки навигации:
	    1. Все сказки. При нажатии пользователь переходит к просмотру всех сказок и возможностью поиска по именам персоонажей и названию сказки
	    2. Недоступно. Не реализованна
	    3. Мои сказки. Список сказок добавленых пользователем, возможность их создания и удаления
	    4. Все викторины. Список всех викторин по сказкам
